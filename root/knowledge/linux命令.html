# linux 命令

# 基础常用

  1. 登录
ssh [用户名]@[IP]
  1. 查看文件夹
ls    # 平铺查看
ll    # 列表查看
ls -a # 查看全部文件，包含隐藏文件
  1. 新建文件夹
mkdir [文件夹名]
  1. 删除文件夹
rm -rf [目录名称]
  1. 修改 移动 文件夹
mv [文件] [目地]
  1. 复制 文件夹
cp [文件] [目的]
  1. 新建文件
touch [文件名]

vi [文件名] # vim 模式新建
vim [文件名] # vim 模式新建
  1. 查看文件
cat [文件名]

head [文件名] # 打印前几行
tail [文件名] # 打印后几行
  1. 查看文件内容
grep ["查找内容"] [文件名]
  1. 学习vim命令
vimtutor
